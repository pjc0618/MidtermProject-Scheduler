Module Task

module Task: sig .. end

A variant that holds each of the updatable properties of any given task.

type updatable_property =
| Description of string
| Completed of bool
| Deadline of float option
| Tags of string list
| Priority of int 
type t

A type representing a task.

val id : t -> int

id task is an arbitrary unique numerical id. No two cards can have the same id.

val description : t -> string

description task is the description of a task. Example: "Do laundry."

val completed : t -> bool

completed task indicates whether the task has already been completed.

val deadline : t -> float option

deadline task is the Unix date and time this task is due wrapped in an option. It is represented as the number of seconds that has passed since 00:00:00 GMT, Jan. 1, 1970.

val tags : t -> string list

tags task is the list of tags bound to task task.

val priority : t -> int

priority task returns the importance of the task on a scale from one to ten, with a 10 being very important/difficult and a one being easy/unimportant

val update_property : t -> updatable_property -> t

update_property task property is task with one of its properties changed with respect to property.

val make : int -> string -> bool -> float option -> string list -> int -> t

make id desc is_done deadline tags priority is a t with the supplied properties.
